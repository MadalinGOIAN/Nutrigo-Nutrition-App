## Nutrigo - Aplicație mobilă de nutriție / Nutrition Mobile App
### RO
Proiectul de licență alcătuit din trei componente:
- bază de date,
- web API,
- aplicație mobilă.

### EN
Final year project with a Three Tier Architecture:
- Database (MySql),
- Web API (C#),
- Mobile App (C#, .NET MAUI Framework)
